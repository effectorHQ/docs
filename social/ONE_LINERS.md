# One-liner value propositions

Tailored for different audiences. Pick the one that fits.

## For MCP developers

> Add a type contract to your MCP tool in 30 seconds. Zero code changes.

## For framework authors

> effector.toml: the tsconfig.json for AI agent capabilities. Declare types once, compile to MCP / OpenAI / LangChain.

## For security-focused teams

> Know exactly what permissions your AI tools need — before they run. `effector-audit` checks declared vs actual behavior.

## For agent builders

> Stop guessing whether two tools can compose. `effector check` tells you before runtime.

## For open source maintainers

> Your MCP server already works. effector.toml adds type safety on top — no breaking changes, no new dependencies.

## For investors / pitch context

> effectorHQ is the type system layer for the AI agent tool ecosystem. Like TypeScript for JavaScript, but for MCP/OpenAI/LangChain tool interfaces. 42 standard types, cross-runtime compilation, zero dependencies.

> AI agent tools lack the type contracts that made APIs reliable. effector.toml is a thin metadata layer that enables static composition checking, permission auditing, and cross-runtime compilation — without changing how tools run.
